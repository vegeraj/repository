# Üdvözlünk a Teendő App dokumentációjában

Ez a dokumentáció azért készült, hogy segítse a **Teendőkezelő v1.0** alkalmazás használatát. Az alkalmazás célja, hogy egyszerűsítse a napi feladatok rendszerezését egyetemi hallgatók számára.

## Mire jó ez az alkalmazás?

* **Rendszerezés:** Kategóriákba sorolhatod a teendőidet.
* **Határidők:** Értesítést kapsz a közelgő beadandókról.
* **Prioritás:** Megjelölheted a sürgős feladatokat.

## Rendszerkövetelmények

Az alkalmazás webes felületű, így telepítést nem igényel.

| Eszköz | Támogatott böngészők |
| :--- | :--- |
| Asztali gép | Chrome, Firefox, Edge |
| Mobil | Chrome (Android), Safari (iOS) |
